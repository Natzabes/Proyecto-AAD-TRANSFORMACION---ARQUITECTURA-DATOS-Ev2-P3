INSTRUCCIONES DE USO - PROYECTO FAMOSOS

1. Abrir la consola de comandos (CMD o Terminal).

2. Tener Python instalado en el equipo.

3. Instalar las librerías necesarias (solo una vez):

   pip install pandas requests pillow

4. Navegar hasta la carpeta del proyecto:

   cd ruta\de\la\carpeta\Registros_Famosos

   Ejemplo:
   cd C:\Users\Usuario\Desktop\Registros_Famosos

5. Ejecutar el programa:

   python main_famosos.py

6. Al abrirse la aplicación, presionar el botón:

   "Cargar CSV"

7. Seleccionar el archivo de famosos que desea procesar. (Use los datos normalizados de la parte 2)

   Ejemplo:
   famosos_normalizados.csv

8. El sistema procesará automáticamente el archivo:

   • Generará el archivo consolidado.
   • Cargará la información en la lista principal.

9. Una vez cargados los datos, aparecerá un listado con el formato:

   • [NOMBRE DEL FAMOSO] - [EDAD] años

   Ejemplo:

   • ABRAHAM LINCOLN - 217 años
   • ALBERT EINSTEIN - 147 años
   • MICHAEL JACKSON - 67 años

10. Para visualizar la información de un famoso:

   • Seleccionar un nombre de la lista.
   • Presionar el botón "Ver información".

11. El visor mostrará:

   • Nombre del famoso.
   • Fotografía obtenida desde Wikipedia.
   • Descripción biográfica.
   • Fuente de la información.
   • Fecha y hora en que se realizó la consulta.

12. Sistema de caché:

   • La primera consulta obtiene los datos desde internet.
   • Las siguientes consultas utilizan el archivo local.
   • Esto reduce el consumo de consultas a la API.

13. Si Wikipedia no dispone de imagen para el famoso:

   • El sistema mostrará el mensaje:
     "Imagen no disponible".

14. Archivos generados por la aplicación:

   Carpeta "outputs"

   • famosos_consolidados.csv

   Carpeta "data"

   • cache_famosos.json

15. Requisitos de conexión:

   • Se requiere acceso a internet para obtener la información de Wikipedia la primera vez.
   • Las consultas posteriores pueden utilizar los datos almacenados en caché.
